# Quick start

Learn how to use effect-playwright with a complete working example.

## Basic example

Here's a simple script that launches a browser, navigates to a page, and prints the page title:

```typescript
import { Playwright } from "effect-playwright";
import { Effect } from "effect";
import { chromium } from "playwright-core";

const program = Effect.gen(function* () {
  const playwright = yield* Playwright;
  const browser = yield* playwright.launchScoped(chromium);
  const page = yield* browser.newPage();

  yield* page.goto("https://example.com");
  console.log(`Page title: ${page.title()}`);
}).pipe(Effect.scoped, Effect.provide(Playwright.layer));

await Effect.runPromise(program);
```

<Tip>
  The `launchScoped` method automatically closes the browser when the scope ends, preventing resource leaks.
</Tip>

## Understanding the code

Let's break down what's happening:

<Steps>
  <Step title="Import dependencies">
    Import the `Playwright` service from effect-playwright, `Effect` from effect, and the browser type from playwright-core:
    
    ```typescript
    import { Playwright } from "effect-playwright";
    import { Effect } from "effect";
    import { chromium } from "playwright-core";
    ```
  </Step>

  <Step title="Create an Effect program">
    Use `Effect.gen` to create a generator-based Effect program:
    
    ```typescript
    const program = Effect.gen(function* () {
      const playwright = yield* Playwright;
      // ...
    });
    ```
  </Step>

  <Step title="Launch a scoped browser">
    Use `launchScoped` to launch a browser that will be automatically closed:
    
    ```typescript
    const browser = yield* playwright.launchScoped(chromium);
    ```
    
    <Note>
      The browser will be closed automatically when the effect's scope ends, even if an error occurs.
    </Note>
  </Step>

  <Step title="Create a page and navigate">
    Create a new page and navigate to a URL:
    
    ```typescript
    const page = yield* browser.newPage();
    yield* page.goto("https://example.com");
    ```
  </Step>

  <Step title="Provide the layer and run">
    Pipe the program through `Effect.scoped`, provide the Playwright layer, and run it:
    
    ```typescript
    await Effect.runPromise(
      program.pipe(Effect.scoped, Effect.provide(Playwright.layer))
    );
    ```
  </Step>
</Steps>

## Managing lifecycle

Using `launchScoped` is the recommended way to manage browser lifecycle. It ensures the browser is closed automatically when the effect's scope ends:

```typescript
const program = Effect.gen(function* () {
  const playwright = yield* Playwright;
  const browser = yield* playwright.launchScoped(chromium);
  // Browser will be closed automatically after this block
}).pipe(Effect.scoped);
```

If you need manual control, use `launch` and close the browser explicitly:

```typescript
const program = Effect.gen(function* () {
  const playwright = yield* Playwright;
  const browser = yield* playwright.launch(chromium);
  
  // ... use browser ...
  
  yield* browser.close;
});
```

<Warning>
  When using `launch` instead of `launchScoped`, you are responsible for closing the browser to prevent resource leaks.
</Warning>

## Browser options

You can pass launch options to customize browser behavior:

```typescript
const browser = yield* playwright.launchScoped(chromium, {
  headless: false,
  slowMo: 100,
  args: ['--start-maximized'],
});
```

## Error handling

All methods return effects that can fail with a `PlaywrightError`. Use Effect's error handling to manage failures:

```typescript
import { Playwright, PlaywrightError } from "effect-playwright";
import { Effect } from "effect";
import { chromium } from "playwright-core";

const program = Effect.gen(function* () {
  const playwright = yield* Playwright;
  const browser = yield* playwright.launchScoped(chromium);
  const page = yield* browser.newPage();

  yield* page.goto("https://example.com");
}).pipe(
  Effect.scoped,
  Effect.provide(Playwright.layer),
  Effect.catchTag("PlaywrightError", (error) =>
    Effect.sync(() => {
      console.error("Browser automation failed:", error.reason);
    })
  )
);

await Effect.runPromise(program);
```

<Note>
  Playwright does not support interruption, so `Effect.timeout` does not behave as you might expect. Use Playwright's built-in `timeout` options instead.
</Note>

## Accessing native Playwright

If you need to access functionality from the underlying Playwright objects that isn't directly exposed, use the `use` method:

```typescript
const program = Effect.gen(function* () {
  const playwright = yield* Playwright;
  const browser = yield* playwright.launchScoped(chromium);
  const page = yield* browser.newPage();

  // Use the native Playwright Page object
  const screenshot = yield* page.use((p) => p.screenshot());
});
```

---
title: Locator
description: Service for interacting with page elements
---

The `PlaywrightLocator` service provides methods for interacting with elements on a page.

## Methods

### click

Clicks the element.

```typescript
click(
  options?: ClickOptions
): Effect.Effect<void, PlaywrightError>
```

<ParamField path="options" type="ClickOptions">
  Optional click options (button, clickCount, delay, position, modifiers, etc.)
</ParamField>

**Returns:** Effect that completes when the element is clicked

### fill

Fills the input field.

```typescript
fill(
  value: string,
  options?: FillOptions
): Effect.Effect<void, PlaywrightError>
```

<ParamField path="value" type="string" required>
  The value to fill in the input field
</ParamField>

<ParamField path="options" type="FillOptions">
  Optional fill options (force, noWaitAfter, timeout)
</ParamField>

**Returns:** Effect that completes when the field is filled

### evaluate

Evaluates a function on the matched element.

```typescript
evaluate<R, Arg = void, E extends SVGElement | HTMLElement = SVGElement | HTMLElement>(
  pageFunction: (element: E, arg: Unboxed<Arg>) => R | Promise<R>,
  arg?: Arg,
  options?: { timeout?: number }
): Effect.Effect<R, PlaywrightError>
```

<ParamField path="pageFunction" type="function" required>
  The function to evaluate on the element
</ParamField>

<ParamField path="arg" type="any">
  Optional argument to pass to the function
</ParamField>

<ParamField path="options" type="object">
  Optional options with timeout
</ParamField>

**Returns:** Effect that resolves to the function's return value

**Example:**

```typescript
import { PlaywrightBrowser } from "effect-playwright";
import { PlaywrightEnvironment } from "effect-playwright/experimental";
import { chromium } from "@playwright/test";
import { Effect } from "effect";

const program = Effect.gen(function* () {
  const browser = yield* PlaywrightBrowser;
  const page = yield* browser.newPage();
  const locator = yield* page.locator("button");
  const buttonContent = yield* locator.evaluate((button) => button.textContent());
}).pipe(PlaywrightEnvironment.provideBrowser, Effect.provide(PlaywrightEnvironment.layer(chromium)));
```

### use

A generic utility to execute any promise-based method on the underlying Playwright `Locator`.

Can be used to access any Locator functionality not directly exposed by this service.

```typescript
use<T>(
  f: (locator: Locator) => Promise<T>
): Effect.Effect<T, PlaywrightError>
```

<ParamField path="f" type="function" required>
  A function that takes the Playwright `Locator` and returns a `Promise`
</ParamField>

**Returns:** Effect that wraps the promise and returns its result

**Example:**

```typescript
const isVisible = yield* locator.use((l) => l.isVisible());
```
